## Password Format

All default passwords follow this pattern:
- **First 4 letters**: Abbreviated section name (e.g., ANTH, MARG, GREG)
- **Next 2 digits**: Grade level (07, 08, 09, 10, 11, 12)
- **Last 2 digits**: Section number within grade (01, 02, 03, etc.)

**Example:** `MARG1201`
- MARG = St. Margaret
- 12 = Grade 12
- 01 = First section in Grade 12

---

## How It Works

### During Registration:

1. **Student selects grade level** (e.g., Grade 12)
   - Password field shows: "Password will auto-fill when you select a section"

2. **Student selects section** (e.g., St. Margaret - STEM)
   - Password field automatically fills with: `MARG1201`
   - Placeholder shows: "Default: MARG1201 (You can change this)"

3. **Student can edit the password**
   - They can keep the default password
   - OR change it to something more private
   - The field is fully editable

### Benefits:

✅ **Easy to remember** - Based on section name and grade
✅ **Quick registration** - No need to think of a password
✅ **Still customizable** - Students can change it if they want
✅ **Consistent format** - Easy for administrators to help students
✅ **Privacy option** - Students can make it more secure

---

## For Students

**Default Password:**
- Your password will automatically appear when you select your section
- You can use this default password or change it to something else
- Make sure to remember your password!

**Changing Your Password:**
- Simply edit the password field during registration
- Or change it later in your profile settings

---

## For Administrators

**Helping Students:**
- If a student forgets their password, you can tell them their section's default
- Example: "Your section is St. Margaret, so your default password is MARG1201"
- They can use this to login and then change it

**Security Note:**
- Default passwords are convenient but not highly secure
- Encourage students to change their passwords for better security
- Especially for students with sensitive information
